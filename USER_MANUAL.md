## 基本概念

本應用程式以**筆記**為核心單元，每則筆記對應一段聖經經文，可以附加：

- **聖經神學標籤**（Biblical Theology）：跨書卷的正典主題，如「約」、「彌賽亞」
- **系統神學標籤**（Systematic Theology）：教義分類，如「基督論」、「救恩論」
- **教義連結**：記錄此段經文如何貢獻於某個教義的具體說明
- **交叉引用**：連結其他相關筆記
- **外部資料**：連結網頁文章、書籍等參考資料

## 新增筆記

1. 點擊左側欄「**新增筆記**」按鈕（或頂部 **+** 按鈕）
2. 選擇**書卷**、**章節**（可選起迄範圍）、**節數**（可選）
3. 輸入**標題**（可選）與**筆記內容**（支援 Markdown 格式）
4. 右上角可選擇**套用範本**快速開始
5. 選擇相關的聖經神學與系統神學**標籤**
6. 加入**教義連結**（選擇教義 + 說明此段經文的貢獻）
7. 點擊「**儲存**」

### Markdown 格式支援

```
# 大標題
## 小標題
**粗體**  *斜體*
- 列表項目
> 引用文字
`行內程式碼`
```

## 瀏覽與搜尋筆記

### 筆記列表

左側欄點擊「**筆記 Notes**」，可依以下方式篩選：
- 書卷
- 聖經神學標籤
- 系統神學標籤

### 全域搜尋

按 **⌘K**（Mac）或 **Ctrl+K**（Windows）開啟全域搜尋，可搜尋標題、內容、經文範圍。

### Dashboard 總覽

首頁顯示：
- 筆記數、標籤數、追蹤鏈數統計
- 聖經進度圖（哪些書卷已有筆記）
- 標籤使用頻率分析

## 經文快速查閱

新增或查看筆記時，頁面會自動顯示「**經文參考**」面板，載入該段的中英對照原文（中文：和合本；英文：WEB 譯本）。

### 顯示模式

面板頂部有三個切換按鈕：

| 按鈕 | 說明 |
|------|------|
| 中文 | 只顯示和合本 |
| 中英 | 中英對照（預設） |
| English | 只顯示 WEB 英文 |

### 節數範圍

- 若筆記指定了**節數範圍**（如 1:1–18），面板預設只顯示選定的節，並以金色高亮標示
- 面板底部有「**顯示整章**」按鈕，可切換查看完整章節
- 點選「**只顯示選擇節數**」可切換回來

### 收合面板

點擊面板右上角的 **∧** 按鈕可收合面板節省空間，再點 **∨** 展開。

> 經文資料來源：bible-api.com（WEB + CUV），結果會快取於伺服器，不需每次重新下載。

## 標籤管理

左側欄點擊「**標籤管理 Tags**」。

### 聖經神學標籤（Biblical）

代表跨書卷的正典主題，預設提供：
約、國度、聖殿、創造與新創造、流放與歸回、律法、獻祭、應許、子民、彌賽亞

### 系統神學標籤（Systematic）

代表神學教義分類，預設提供：
神論、基督論、聖靈論、人論、罪論、救恩論、教會論、末世論、聖經論、贖罪

### 新增 / 編輯 / 刪除標籤

- 點擊「**+ 新增標籤**」輸入名稱並選擇顏色
- 點擊現有標籤可編輯名稱或顏色
- 刪除標籤時，所有使用該標籤的筆記會自動移除該標籤

## 系統神學教義連結

在筆記編輯頁底部的「**系統神學教義連結**」區塊：

1. 點擊「**+ 連結系統神學教義**」
2. 從下拉選單選擇教義分類
3. 填寫**註解**：說明此段經文如何具體貢獻於此教義
   - 例：「逾越節羔羊的替代性死亡，預表基督代贖的核心邏輯」
4. 點擊「**新增**」

在「**教義 Doctrines**」頁面可以從教義的角度，查看所有連結到某個教義的經文與筆記。

## 主題追蹤鏈

追蹤一個神學主題從舊約到新約的發展脈絡。

### 建立追蹤鏈

1. 點擊左側欄「**追蹤鏈 Chains**」
2. 點擊「**+ 新增追蹤鏈**」
3. 輸入名稱、說明，選擇對應的聖經神學標籤
4. 加入相關筆記（按正典順序排列）

### 使用方式

追蹤鏈會以時間軸方式呈現各筆記，幫助觀察主題在救贖歷史中的發展。

## 交叉引用

在筆記詳情頁的「**交叉引用**」區塊：

1. 點擊「**+ 引用筆記**」
2. 搜尋並選擇目標筆記
3. 填寫引用說明（可選）
   - 例：「此處的羔羊意象與出埃及記 12 章互相呼應」
4. 點擊「**建立引用**」

引用分為「→ 引用」與「← 被引用」兩個方向，點擊可直接跳轉到被引用的筆記。

## 外部資料管理

### 新增資料來源

左側欄「**外部資料 Resources**」頁面，支援三種類型：

| 類型 | 適用場景 |
|------|---------|
| 🌐 網址 URL | 網路文章、講道錄音、線上工具 |
| 📚 書籍 Book | 神學書籍、注釋書 |
| 📄 其他 Other | 論文、私人筆記 |

### 連結到筆記

在筆記詳情頁的「**外部資料**」區塊，點擊「**+ 連結資料**」，從已建立的資料庫中搜尋並連結。

## AI 輔助分析

在筆記詳情頁底部的「**AI 輔助分析**」區塊提供三種功能：

| 按鈕 | 功能 |
|------|------|
| 建議相關經文 | 根據筆記內容，建議 5–8 段主題相近的其他聖經段落 |
| 建議教義連結 | 分析筆記可連結的系統神學教義，並說明具體貢獻 |
| 延伸研究方向 | 建議下一步可深入探索的研究方向與問題 |

AI 分析結果會考慮你已有的標籤分類，讓建議能對應到系統中的教義架構。

---

### 啟用 AI 功能

AI 功能需要 **Anthropic API Key**。取得方式：前往 https://console.anthropic.com 註冊並建立 API Key（格式：`sk-ant-...`）。

#### 重新啟動容器並加入 API Key

SSH 進 NAS，執行：

```bash
sudo docker rm -f bible-notebook && \
sudo docker run -d \
  --name bible-notebook \
  -p 3000:3000 \
  -v /volume1/docker/bible-notebook/data:/app/backend/data \
  -e ANTHROPIC_API_KEY=sk-ant-你的金鑰 \
  --restart unless-stopped \
  bible-theology-notebook
```

啟用後，AI 輔助面板的按鈕會變為可點擊狀態。

---

### 停用 AI 功能

不傳入 API Key 重新啟動容器即可停用：

```bash
sudo docker rm -f bible-notebook && \
sudo docker run -d \
  --name bible-notebook \
  -p 3000:3000 \
  -v /volume1/docker/bible-notebook/data:/app/backend/data \
  --restart unless-stopped \
  bible-theology-notebook
```

停用後，AI 輔助面板會顯示灰色提示，其他所有功能不受影響。

---

### 確認 AI 功能狀態

瀏覽器開啟：http://192.168.50.8:3000/api/ai/status

- `{"hasKey":true}` → AI 已啟用
- `{"hasKey":false}` → AI 未啟用

## 資料備份與還原

### 匯出備份

左側欄底部點擊「**匯出備份**」，下載 JSON 格式備份檔，包含所有筆記、標籤、追蹤鏈、教義連結、外部資料。

### 匯入還原

點擊「**匯入還原**」，選擇備份的 JSON 檔案，確認後覆蓋現有資料。

> **注意**：匯入會覆蓋全部現有資料，請先匯出備份。

### 直接備份資料庫檔案

更完整的備份方式是直接複製 SQLite 資料庫：

```bash
cp /volume1/docker/bible-notebook/data/notebook.db \
   /volume1/docker/bible-notebook/data/notebook.db.backup
```
